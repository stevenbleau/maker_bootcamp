# Intro to Vector Design — Build Plan

Goal: create a Quarto RevealJS deck for Intro to Vector Design using the same layout/theme conventions as Intro to CAD.

## Starter checklist

- Replace section divider image at `images/0/0.0.0.png`
- Add step images with naming `{section}.{step}.{image_index}.png`
- Edit `deck.qmd` for slide content
- Adjust layout/colors in `theme/maker-bootcamp.scss` if needed
- Run `quarto preview deck.qmd` while authoring

## Conventions

- Main screenshots should be full app-window captures in 16:10 or 16:9
- Camera images are separate from screenshots and should use a plain/neutral background
- For 3D objects/photos, prefer view-cube-like orientations (top/left/right/front/isometric)
