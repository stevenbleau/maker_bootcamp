# Image Naming

Use: `{section}.{step}.{image_index}.png`

Examples:
- `0.0.0.png` section divider image
- `0.1.0.png` main screenshot for step 0.1
- `0.1.1.png` callout image for step 0.1

Keep camera photos separate from UI screenshots by suffixing with `.cam` when needed, for example:
- `2.3.0.cam.png`
