# Make Track

This folder is for maker tool tutorials and operational workflows.

Planned examples include:

- 3D printers
- slicing software
- MIDI controllers
- camera use
- laser cutters

Recommended module pattern:

```text
Make/
└── Intro to <Tool>/
    ├── _quarto.yml
    ├── theme/
    └── decks/
        └── 01-<topic>/
            └── deck.qmd
```
