# Create Track

This folder is for project-building and creative application modules.

Use this track for modules that combine tools and skills into finished outputs
(for example: capstone builds, creative projects, and production workflows).

Recommended module pattern:

```text
Create/
└── <Project Module>/
    ├── _quarto.yml
    ├── theme/
    └── decks/
        └── 01-<topic>/
            └── deck.qmd
```
